# Create chart

This step is one of the three outputs. It allows you to chart and customize a dataset.

![image.png](https://alphacast-images.s3.us-west-2.amazonaws.com/next-s3-uploads/6b381b0b-2a1c-4230-adab-b65c1c4147e8/image.png)

To create a new basic chart based on the data of the pipeline up to that point you have to select the step "Create a chart" and save. The pipeline will make you select which variable you want to chart, you can select them from the pipeline you had done or add a variable from a other dataset:

![image.png](https://alphacast-images.s3.us-west-2.amazonaws.com/next-s3-uploads/f99ed456-9b81-4588-b482-020301d989e9/chart-1-r.png)

Once you select the diferent variables you want to chart, the pipeline will ask you to configure it:

![image](https://alphacast-images.s3.us-west-2.amazonaws.com/next-s3-uploads/0e5491e8-9a22-44f9-9dec-7bf93eabb403/chart-2-r.png)


To customize it use the diferents tabs: Basic, Data, Texts, Customize and Annotations. Each one has diferent funtions, click [here](https://www.youtube.com/watch?v=7IU_o6WGLmI&t=1s) for more information.
