# Filter Rows 

This step allows you to filter by rows, whether you want to keep rows from a specific date or with rows where values meet a criteria

![filter rows 1](https://user-images.githubusercontent.com/94381642/220921979-243ad33e-9bef-4e2e-b068-d4d7e4191d0a.png)


This criterion can be a logical function, an operation and even comparisons that allow us to eliminate the values that we are interested in seeing from a series of data

![Filter rows 2](https://user-images.githubusercontent.com/94381642/220922040-1defdc7a-921a-4229-a5cd-981ff866f7b0.png)

There are dozens of operators, comparisons, and functions that allow endless combinations. For a comprehensive list of functions and operands see [here](https://www.alphacast.io/p/luciano/insights/2022-2-1-pOLoak)
