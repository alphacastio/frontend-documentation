# Filter Rows 

This step allows you to filter by rows, whether you want to keep rows from a specific date or with rows where values meet a criteria

![filter rows 1](https://alphacast-images.s3.us-west-2.amazonaws.com/next-s3-uploads/4c927323-399e-4b2b-be4d-5513614caf7e/filter-row-primer.png)


This criterion can be a logical function, an operation and even comparisons that allow you to eliminate the values that we are interested in seeing from a series of data

![Filter rows 2](https://alphacast-images.s3.us-west-2.amazonaws.com/next-s3-uploads/7e34f959-737f-4c8b-8c53-04bd0758826e/Filter-row-1-posta.png)

There are dozens of operators, comparisons, and functions that allow endless combinations. For a comprehensive list of functions and operands see [here](https://www.alphacast.io/p/luciano/insights/2022-2-1-pOLoak)
