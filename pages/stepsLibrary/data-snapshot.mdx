# Data Snapshot

This step is one of the three outputs. This step takes a snapshot of the pipeline up to that point

![image](https://user-images.githubusercontent.com/90727312/222264681-6da0df69-e2b0-4a19-9c2a-c8092c0aadbc.png)

It provides a link to download the data, a code line to embed it on a website or an insight, and (optionally) an excerpt on the pipelines result. Links may be available in several different formats: CSV, XLSX, or TSV.
