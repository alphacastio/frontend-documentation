# Data Snapshot

This step is one of the three outputs. This step takes a snapshot of the pipeline up to that point

![image.png](https://alphacast-images.s3.us-west-2.amazonaws.com/next-s3-uploads/6b381b0b-2a1c-4230-adab-b65c1c4147e8/image.png)

It provides a link to download the data, a code line to embed it on a website or an insight, and (optionally) an excerpt on the pipelines result. Links could be in CSV, XLSX or TSV format
