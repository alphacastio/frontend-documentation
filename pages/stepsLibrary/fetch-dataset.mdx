# Fetch Dataset

This step allows you to choose the dataset to which the sequence of pipeline steps can be added. 

![image](https://user-images.githubusercontent.com/90727312/222451845-741db79a-ca1e-4a34-a6ab-790a62bd57d1.png)


This step is already predefined and is usually the first step of a pipeline

To select your dataset start writing in the search box and recommendations will show up.
